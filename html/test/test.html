Pokegestion

About

This project is a simple Pokédex in HTML/CSS & JavaScript. This is the first part of the project, implementing the Pokemon, Attack and Type classes and their methods.
This page allows you to test the methods of the classes implemented in the first part of the project.

How to use

Press [F12] to open the console
Enter a parameter in the input field below and click on a test to see the result in the console and below.

Here are some examples of parameters you can enter:

  • "Bug" for the type
  • "Stone Edge" for the attack
  • "Pikachu" for the Pokémon
Be careful, the parameters are case sensitive!

Tests


  1. Get the list of Pokémon by type
  2. Get the list of Pokémon by attack
  3. Get the list of attacks by type
  4. Get the list of Pokémon sorted by name
  5. Get the list of Pokémon sorted by decreasing stamina
  6. Get the list of Pokémon for which the chosen attack is the most effective
  7. Get the list of the most effective attack types against a given Pokémon
